## 3. 사용자가 직접 해야 할 작업 (외부 등록)

### 3-1. 구글 서치콘솔 색인 진단 ⭐ 최우선

1. https://search.google.com/search-console 접속
2. 좌측 메뉴 **색인 → 페이지** 진입
3. **"색인 미생성" 항목**을 확인:
   - 차단된 페이지가 있는지
   - 중복 또는 표준 페이지로 선택되지 않음 — doorway 의심 신호일 수 있음
   - 크롤링됨 - 현재 색인되지 않음 — 콘텐츠 품질 신호 부족
4. **사이트맵 제출 확인**: 좌측 메뉴 **색인 → Sitemaps**
   - `https://simplecube.net/sitemap-index.xml` 등록 확인
   - `https://simplecube.net/sitemap-blog.xml` 추가 등록
5. **URL 검사** 도구로 핵심 페이지 5개 강제 색인 요청:
   - `/rental/`
   - `/wedding/`
   - `/popup/`
   - `/seoul/`
   - `/busan/`
6. 이번 브랜치 배포 후 위 페이지들 다시 **"색인 생성 요청"** 실행

### 3-2. 네이버 서치어드바이저 (웹마스터도구)

1. https://searchadvisor.naver.com/ 접속
2. 사이트 등록 확인 (이미 메타 태그는 들어감: `99c64b13d13f...`, `150bf729f3a9...`)
3. 좌측 **요청 → 웹페이지 수집** 으로 핵심 페이지 5~10개 강제 색인 요청
4. **사이트맵 제출**:
   - `https://simplecube.net/sitemap-index.xml`
   - `https://simplecube.net/sitemap-blog.xml`
5. **간단 분석 → 최적화 보고서** 확인 — 누락된 메타 태그, robots 차단, 모바일 친화도 점검
6. **기능 → robots.txt 검증** — 의도치 않게 차단된 경로 없는지 확인

### 3-3. 네이버 플레이스 등록 ⭐ 매우 중요

자연검색에서 가장 큰 손해. 등록 안 된 상태로 추정.

1. https://smartplace.naver.com/ 접속
2. **신규 등록** → 업체 정보 입력:
   - 업체명: `심플큐브 포토부스` (정확히)
   - 카테고리: 사진관 / 이벤트 대행 / 결혼식 서비스 (다중 선택)
   - 주소: 서울 본사 + 부산·대전·전주 지점 각각 (지점별 별도 등록 가능)
   - 전화번호: 02-338-9180 (본사), 010-3122-4746 (부산), 010-8838-8122 (대전), 010-5743-8122 (전주)
   - 영업시간: 평일 10:00–18:00
   - 홈페이지: https://simplecube.net
3. **사진 5장 이상 등록** (업체 외관, 부스 작품 사진, 행사 사례)
4. **소개글에 키워드 포함**: "포토부스 대여", "웨딩 포토부스", "팝업 포토부스" 자연스럽게 삽입
5. **사업자등록증·통신판매업 신고증** 업로드 → 인증 마크 부여 (신뢰도 큰 영향)
6. 등록 후 1~2주 내 노출 시작. 리뷰가 쌓이면 순위 상승.

### 3-4. 네이버 블로그 후기 자산 만들기

VIEW 영역 노출이 0건. 자체 블로그 콘텐츠 또는 협업 후기 필요.

1. **자체 블로그**: 네이버 블로그 별도 운영하지 않더라도 브랜드 블로그 1개 생성
   - 주소: `blog.naver.com/simplecube_photobooth` 형태
   - 1주 1포스트씩 — 진행한 행사 후기 (사진 + 짧은 본문)
2. **고객 후기 요청**: 행사 종료 후 신랑신부·기업 담당자에게 후기 요청
   - 카카오톡 후기 자동 발송 시 네이버 블로그 후기 작성 시 다음 행사 5% 할인 같은 인센티브
3. **공식 카페 활동**: 결혼준비 카페(레몬테라스, 미즈넷 등)에 설치 사례 공유

### 3-5. 브랜드 보호 (쿠팡 얼음틀 침범 대응)

`심플큐브` 검색 시 쿠팡 "심플큐브 얼음틀" 광고가 침범 중.

1. **네이버 검색광고**: 본인 브랜드 키워드 `심플큐브`, `simplecube`, `심플큐브 포토부스`에 본인 사이트 광고 입찰 (CPC 낮음, 방어용)
2. **구글 광고**: 동일 — 브랜드 키워드 방어 광고
3. **장기적 대응**: 자체 콘텐츠 강화 — 구글 검색 결과의 첫 4~5개를 자사 페이지로 채우기 (현재 진행 중)

## 6. 코드 측면 추가 개선 후보 (지금 적용 안 된 것)

향후 개선 여지가 있는 부분:

1. **블로그 글 32개 색인 누락 확인** — 사이트맵에는 있는데 실제 색인 안 되어 있으면 콘텐츠 품질·내부 링크 부족 가능성. 서치콘솔에서 개별 URL 검사 필요.
2. **`/[...slug]` 동적 라우트 점검** — 신규 추가된 catch-all 라우트가 의도치 않은 페이지를 색인하지 않는지 검증
3. **OG 이미지 최적화** — 현재 `/images/photobooth-rental-simplecube.jpg` 1.2MB. 800x630 webp로 변환하면 SNS 공유 시 더 빠름
4. **블로그 카테고리/태그 페이지 추가** — VIEW 영역 노출 강화. 현재는 개별 글만 있음
5. **JSON-LD `LocalBusiness` 다중 지점화** — 현재 본사 1곳만. 부산·대전·전주 지점도 별도 LocalBusiness schema로 추가하면 지역 검색 신호 강화
